Robotics Links

This is a mostly-unformatted list of cool robotics stuff, whether it be books, components or robots themselves. I plan to keep this up-to-date as I find items of interest.

Robots

  • SCAMP Really sweet quadcopter that, using micro-hooks, can attach to and climb up a wall
  • Bat bot Bat robot. 'Nuff said.
  • ARTI3 Passivly articulated tracked robot for stair climbing
  • High precision teleoperation From UT Austin, a cool control interface
  • Panasonic assist exoskeletons Title says it all
  • Shape memory metal foam for soft robotics
  • SRI's PCB-based microrobots
  • Non-holonomic walking robot
  • Google has a group of robots that share training data to speed up CNN-based learning
  • UPenn Picobug-tiny walking flying robot. Cool parts include origami walking linkage, and using quadcopter to control rotation when walking

Resources

  • Seminar on how weirdly good human performance is
  • IEEE Spectrum Robotics tag
  • Talk on challenges in drone flight
  • CMU robotics youtube channel
  • Free ebook from CSAIL on underactuated robotics

Components

  • Fast, high-precision IR sensor for drones
